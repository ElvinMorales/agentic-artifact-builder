# Agentic Artifact Builder

An interactive educational builder for creating agentic AI artifacts from a framework-neutral taxonomy.

This project is a companion to the Agentic AI Artifact Taxonomy. The taxonomy defines the conceptual model; this builder helps people learn the model by filling guided forms and generating reusable artifacts such as `SKILL.md`.

## MVP

The first version will focus on a guided `SKILL.md` builder:

- Fill in capability information
- Preview generated Markdown
- Copy or download the generated artifact
- Learn how the artifact maps back to the taxonomy

## Public-safety note

This repo uses synthetic, generic examples only. Do not include employer-specific workflows, secrets, private data, unsanitized logs, or proprietary examples.
